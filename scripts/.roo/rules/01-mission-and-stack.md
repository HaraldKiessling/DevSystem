# Mission & Technischer Stack

## Ziel
Aufbau eines reproduzierbaren, cloudbasierten Entwicklungssystems auf einem IONOS Ubuntu VPS. Das System muss vollständig per Handy-Browser (PWA) über code-server steuerbar sein.

## Technischer Stack
- **OS:** Ubuntu (IONOS VPS).
- **Netzwerk:** Tailscale VPN (einzigzugelassener Zugangsweg).
- **Proxy:** Caddy (HTTPS/SSL Management).
- **IDE:** code-server (VS Code im Browser).
- **KI-Steuerung:** Roo Code (Multi-Agent).
- **KI-Provider:** OpenRouter (Cloud) & Ollama (Lokal auf VPS).

## Zugriff
Der Zugriff erfolgt ausschließlich verschlüsselt über die Tailscale-IP des Servers.
